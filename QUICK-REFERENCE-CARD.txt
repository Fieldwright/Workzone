┌─────────────────────────────────────────────────────────────────────────────┐
│  HOW TO FIND & EDIT CODE                                                    │
└─────────────────────────────────────────────────────────────────────────────┘

  1. Know what you want to change → Pick the PART number above
  2. Open the corresponding file
  3. Search for: "PART X - START"  (where X is your part number)
  4. Edit the code between START and END markers
  5. Save and refresh browser

  ⚠️  NEVER DELETE THE START/END MARKERS! ⚠️

┌─────────────────────────────────────────────────────────────────────────────┐
│  SEARCH COMMANDS                                                            │
└─────────────────────────────────────────────────────────────────────────────┘

  In your editor:
    Ctrl+F (Windows) or Cmd+F (Mac) → "PART X - START"

  In terminal/command line:
    grep -n "PART 7" calculations.js

  Find a function:
    Ctrl+F → "function nameYouWant"

┌─────────────────────────────────────────────────────────────────────────────┐
│  EMERGENCY: Something Broke?                                                │
└─────────────────────────────────────────────────────────────────────────────┘

  1. Open browser console (F12)
  2. Look for red errors
  3. Error shows file name and line number
  4. Go to that PART in that file
  5. Check for missing brackets } or semicolons ;
  6. Verify START/END markers are intact
  7. If stuck, revert to backup
